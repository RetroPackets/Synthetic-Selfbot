# Installation & Usage
**Clone from Github**:
```
$ git clone https://github.com/rly0nheart/thedevilseye.git
```

```
$ cd thedevilseye
```

```
$ pip install -r requirements.txt
```

```
$ python eye search-query
```

**Install from Pypi**:
```
$ pip install thedevilseye
```

```
$ eye search-query
```

**Upgrade to latest version**:
```
$ python eye --update
```

```
$ pip install --upgrade thedevilseye
```
